ЧИСЛЕННОЕ РЕШЕНИЕ ЛОКАЛИЗОВАННОЙ ОБРАТНОЙ ЗАДАЧИ ЭЛЕКТРОКАРДИОГРАФИИ

1. Введение 
В традиционной постановке обратная задача электрокардиографии представляет собой задачу определения потенциала на внешней поверхности сердца по измерениям потенциала на поверхности грудной клетки (см., например, [1], [2]). Одним из наиболее важных применений методов решения этой обратной задачи является диагностика очагов аритмии сердца. 
В работе [3] были предложены алгоритмы решения обратной задачи электрокардиографии для модельной геометрии торса и сердца, в [4] рассматривалась реальная геометрия торса и сердца, но предполагалось, что грудная клетка однородна, в [5, 6, 7] рассмотрена обратная задача электрокардиографии с учетом внутренней неоднородности грудной клетки, а в [8] был предложен алгоритм определения проекции очага аритмии на внешнюю поверхность сердца. Следует отметить, что во всех указанных выше работах решалась обратная задача определения потенциала электрического поля и локализации очага армитмии только на внешней поверхности сердца.
Цель данной работы состоит в том, чтобы показать возможность локализации очага аритмии в результате определения потенциала не на внешней поверхности сердца, а на поверхности одной из выделенных частей сердца, в которой предположительно находится источник аритмии.

3. Результаты численных экспериментов 
Рассмотрим результаты численных экспериментов для реальной геометрии торса и сердца. Поверхность торса ΓB и внешняя поверхность сердца ΓH были реконструированы по данным компьютерной томографии. Число граничных элементов на поверхности ΓB составило 2784, на поверхности ΓH – 3082. Далее поверхность сердца разбивалась на три части, соответствующие анатомическим частям сердца:
• ΓLV представляла собой поверхность левого желудочка (Left Ventricle), число граничных элементов составило 3120;
• ΓSP представляла собой поверхность межжелудочковой перегородки (Septum), число граничных элементов составило 3056;
• ΓRV представлял собой поверхность правого желудочка (Right Ventricle), число граничных элементов составило 3014.
С использование программного обеспечения Oxford Cardiac Chaste [15] выполнялась процедура т.н. виртуального пейсинга из левого и правого желудочков, заключающаяся в численном моделировании электрического поля сердца, создаваемого в результате электрической стимуляции участка миокарда в левом и правом желудочках. В результате численного моделирования реконструировались значения потенциала электрического поля сердца на поверхности торса. В эти данные вносилась погрешность d = 3%.
Далее с этими данными при помощи изложенного выше алгоритма решались задачи для каждой из поверхностей ΓLV , ΓSP ,ΓRV и анализировались невязки решения на поверхности торса. Значения невязок составили.
Для источника в области OLV минимальная невязка соответствовала решению для ΓLV , а для источника в области ORV минимальная невязка соответствовала решению для ΓRV . Таким образом, изложенный в работе численный алгоритм позволяет при достаточно хорошей точности измерений достоверно определить поверхность одной из выделенных частей сердца, содержащую источник электрического поля.

Обратная задача электрокардиографии является задачей определения потенциала на внешней поверхности сердца по измерениям потенциала на поверхности грудной клетки. Решение задачи используется при диагностировании очагов аритмии. В математической модели электрическое поле сердца определяется одним источником в сердечной мышце. Рассматриваются поверхности ΓLV (левый желудочек), ΓSP (межжелудочковая перегородка) и ΓRV (правый желудочек), при этом источник электрического поля находится внутри одной из поверхностей. Требуется найти функцию u(x), представленную некорректно поставленной задачей Коши для уравнения Лапласа. При помощи метода Тихонова ищется решение дискретного аналога операторного уравнения обратной задачи. С помощью Oxford Cardiac Chaste численно моделировалось электрического поля сердца. Анализировались невязки решения описанной задачи. Алгоритм позволяет достоверно определять поверхность сердца, содержащую источник электрического поля.
